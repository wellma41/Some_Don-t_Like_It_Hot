A FIPS code is a 5 digit code used in the untied states to denote unique counties. The first 2 digits (left filled with a 0 for single digit codes) represent the state, for instance 26 refers to Michigan. The trailing 3 digits refer to the county within that state, for instance 163 is for Wayne County, where Detroit is.
